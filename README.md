# ArchDroidOS-Project
ArchDroidOS Project is a Virtual Machine in Virtual Master, with Android 5.1.1. It runs 100% without damaging your actual device, and yes, the name is familiar. It comes with MiXplorer, SuperUser (ClockWork Mod), Material Terminal, and can even run light games.
